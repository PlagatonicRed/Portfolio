Sterile Coffin

Sterile Coffin is a short, first person, movement shooter, set in an abandoned laboratory. As the player you must escape from this unknown and unnatural environment by fighting your way through waves of enemies.

Development Time: 6 weeks

Team Size: 5

Goals:
Create designer friendly tools
learn how to create a secure save/load system
Create clean and reusable scripts.

Role: Programming Lead


My contributions to Sterile Coffin are primarily programming related, as I was repsonsible for game systems, player controls & movement, audio and animation integration, and Enemy AI. Though out of all of these I feel as if I am most proud of the systems I created for the game.


I also went into this project with two main things that I wanted to improve on by the end of developement. Firstly, I wanted to integrate uml design theory into my game development process and also generally take more time to make proper documentation for any scripts that I make. Secondly, I wanted learn how to use Unity's custom GUI tools to develop more designer friendly tools that could speed up production.
